## Installation


```bash
    # Clone the repository #
    git clone https://github.com/ashtonqlb/vlbo.git
    # Install dependencies #
    bun install
    # Run the TailwindCSS generation script before launching the server #
    bun css:linux

    # The server's default port is 8080 #
    bun start
```
