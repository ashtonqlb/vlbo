Available to view on [Cyclic](https://combative-leotard-elk.cyclic.app/)

## Installation

1. Clone this repository
   
    ``` bash
   git clone https://github.com/ashtonqlb/vlbo.git
    ```
2.  Run 
    ``` bash
    bun install
    ```
